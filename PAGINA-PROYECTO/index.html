Informacion

Este proyecto lo pensamos para poder solucionar un problema a aquellas personas las cuales tienen el gran problema visual como lo es la ceguera, para solucionarlo decidimos hacer unos lentes los cuales tendran unos sensores que sonaran de una manera u otra dependiendo si tiene un objeto u obstaculo enfrente. Este proyecto lo hemos realizado en años anteriores esto nos proporciono encontrarle fallas o incomodidades y tambien como solucionarlas, un ejemplo claro de esto es el soporte, anteriormente utilizariamos una caja la cual tendria que sostener el usuario por medio de una "cuerda" esto proporcionaria una incomodidad a la hora de utilizar este proyecto. Asi que en este modelo 2.0 realizamos cambios a este soporte brindando comodidad y ayuda al usuario.


Componentes

Para la resolucion del proyecto primero tenemos que saber que materiales utilizar para cumplir nuestro objetivo, para esto necesitaremos unos anteojos preferentemente de sol, luego tambien unos sensores ultrasonicos los cuales nos van a poder detectar la distancia la cual se encuentra el obstaculo, tambien un buzzer el que sonara de una manera u otra dependiendo de que tan lejos se encuentre nuestro objetivo y por ultimo pero no menos importante la ayuda de una bateria de 9 Voltios para poder brindarle una energia a la placa sin necesidad de que este conectada a la computadora.
